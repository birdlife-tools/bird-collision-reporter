# Bird Collision Reporter

Lightweight PWA for reporting bird-glass collisions with GPS coordinates.

## Problem

Collisions with glass facades in cities kill billions of birds annually. Organizations need coordinated citizen reports to locate "black spots" and pressure developers to install bird-safe films.

## Solution

Progressive web app optimized for mobile phones. User photographs a collision victim in three clicks, app captures GPS coordinates and sends anonymous report.

## Technical Stack

- HTML5 Geolocation API
- Edge AI (TensorFlow.js on client for basic "is this a bird" verification)
- Offline mode for caching data without signal
